# Email Validator Service

Сервис проверки валидности email-адресов с использованием SMTP-пробника, отправки тестового письма и анализа bounce-сообщений.  
FastAPI + Redis + SQL Server.
Скоро прикреплю реализацию в docker и ci, так как планируется разбиение на 2 образа: апи-сервис и сервис отправки

## Компоненты

- **FastAPI** – принимает запросы, сохраняет задачу в БД, отправляет `record_id` в Redis.
- **Валидатор** – фоновый процесс, слушает Redis, выполняет проверки, рассылает письма и обновляет БД.
- **Redis** – очередь задач и хранилище ожидающих bounce.

## API

- **POST /send** – cоздаёт задачу на проверку email.
 Header: Authorization: Bearer <token>
 Body: {
  "email": "user@example.com",
  "subject": "Тест",
  "body": "Привет, это тестовое письмо"
}
  Success response(200):
  {
  "success": true,
  "queueId": "XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX"
}

Errors:
- 400 – email уже в очереди или невалидные параметры.
- 401 – неверный токен.
- 500 – внутренняя ошибка.
